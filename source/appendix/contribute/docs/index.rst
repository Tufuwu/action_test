Dokumentation
=============

Die Dokumentation wird bei github gelagert und bei Transifex
übersetzt. Es ist sehr einfach, hier ohne weitere Hilfsmittel
mitzuhelfen.

Ein bisschen herausfordernder ist, die Dokumentation in größerem
Rahmen zu überarbeiten und lokal zu bauen, aber auch dafür gibt es
eine Anleitung.
